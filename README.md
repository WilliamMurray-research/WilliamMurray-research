# William Murray

**See the universe in a grain of sand**

I build frameworks for understanding how structure gives rise to behaviour.  
My research explores non‑backtracking dynamics on mesoscale networks and the
conditions under which Tier‑1 emergence appears.  
I publish the methods, the experiments, and the results - nothing more.

---

## Current Work

- **Tier‑1 Emergence** - a framework for studying emergent behaviour in structured graphs  
- **Experiment 1** - fixed parameters, searching for stable motifs and metastability  
- **Experiment 2** - parameter variation and phase structure mapping  
- **Open papers** - reproducible methods, figures, and replication instructions
